### 2. Предобработка и очистка данных - до 2 баллов

- [x] Даты приведены к типу `datetime`.
- [x] Курсы валют преобразованы из строк в числа.

- [x] Сформирован отчет по пропускам.
- [x] Объяснены естественные пропуски в расчетных признаках.
- [x] Найдены экстремальные дневные изменения через Z-score.
- [x] Созданы новые расчетные признаки.

Созданные признаки:

- `daily_return_pct` - дневное процентное изменение.
- `log_return` - логарифмическая доходность.
- `rolling_mean_30` - 30-дневная скользящая средняя.
- `rolling_volatility_30` - 30-дневная годовая волатильность.
- `rate_index_2014` - индекс курса от начала 2014 года.
- `extreme_return` - флаг экстремального дневного изменения.
- `year`, `month`, `quarter`, `period` - временные признаки.

Как это выполнено:
